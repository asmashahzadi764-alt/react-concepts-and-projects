## 📚 Topics Covered

### 🔹 What are Components?

Components are the building blocks of a React application. They allow developers to divide the user interface into small, independent, and reusable pieces, making applications easier to develop and maintain.

### 🔹 Functional Components

A Functional Component is a JavaScript function that returns JSX. Modern React applications primarily use functional components because they are simple, reusable, and support React Hooks.

### 🔹 Creating Components

Learned how to create custom React components in separate `.jsx` files to keep the project organized and modular.

### 🔹 Importing and Exporting Components

Understood how components are exported using `export default` and imported into other files using the `import` statement.

### 🔹 Rendering Multiple Components

Learned how multiple components can be combined inside the root `App` component to build a complete application.

### 🔹 Component Hierarchy

Explored how the React application follows a parent-child hierarchy where the `App` component acts as the root component and renders other child components.

### 🔹 Reusable Components

Learned that a single component can be rendered multiple times without duplicating code, improving maintainability and reducing repetition.

### 🔹 Component Folder Structure

Organized components inside a dedicated `components` folder to keep the project clean and scalable.

### 🔹 Component Naming Convention

Learned that React component names should always begin with an uppercase letter (PascalCase) so React can identify them as custom components.
