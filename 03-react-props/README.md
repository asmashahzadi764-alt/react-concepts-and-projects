# 📚 Topics Covered

## 🔹 What are Props?

Props (short for *Properties*) are used to pass data from a parent component to a child component. They make components reusable by allowing different data to be displayed using the same component.

---

## 🔹 String Props

String values were passed from the parent component to child components to display different course titles and descriptions.

**Example concepts covered:**

- Passing text values
- Accessing props inside a component
- Reusing the same component with different data

---

## 🔹 Number Props

Numeric values were passed through props to display course prices.

**Example concepts covered:**

- Passing numbers using curly braces
- Difference between strings and numbers in JSX

---

## 🔹 Boolean Props

Boolean values (`true` and `false`) were used to display different course availability statuses.

**Example concepts covered:**

- Conditional rendering
- Ternary operator
- Dynamic UI based on boolean values

---

## 🔹 Object Props

Instead of passing multiple individual props, a complete object was passed to the component.

**Example concepts covered:**

- Passing objects
- Accessing nested object properties
- Organizing related data

---

## 🔹 Array Props

Arrays were passed as props and rendered dynamically using JavaScript's `map()` method.

**Example concepts covered:**

- Passing arrays
- Rendering lists
- Using `key` inside `map()`

---

## 🔹 Function Props

Functions were passed from the parent component to child components and executed when a button was clicked.

**Example concepts covered:**

- Passing functions as props
- Event handling
- Parent-to-child communication

---

## 🔹 Props Destructuring

Props destructuring was introduced to write cleaner and more readable React components.

**Example concepts covered:**

- Object destructuring
- Cleaner component syntax
- Reducing repetitive `props.` usage

# 🎯 Learning Outcomes

After completing this topic, the following concepts were understood:

- Passing data between React components
- Making components reusable
- Passing different data types through props
- Rendering lists using array props
- Passing functions between components
- Using conditional rendering with boolean props
- Organizing data with object props
- Writing cleaner code using props destructuring

---

# 🚀 Key Takeaways

- Props provide one-way data flow from parent to child components.
- Components become reusable when different data is passed through props.
- Props can store strings, numbers, booleans, objects, arrays, and functions.
- Array props are commonly rendered using the `map()` method.
- Function props enable communication between components through events.
- Destructuring improves readability and reduces repetitive code.
